# hackcast

MicroEletronics Native MBR for Embedded Linux on NandFlash" devices.

Commercial device names:
- MiraScreen M2 Pro (sage_8268B_mirascreen-dongle_8Mkey-8723as)
- AnyCast M2 Plus (am_8268B_anycast-dongle_8MFree)
